# BountyHub Portal

A comprehensive platform for managing bounties, community engagement, and real-time communication.

## Features

### Real-Time Chat System
- **Live Message Updates**: Messages appear in real-time without page refresh
- **Efficient Polling**: Smart polling mechanism that only fetches new messages
- **Page Visibility Detection**: Automatically pauses updates when tab is not visible
- **Desktop Notifications**: Browser notifications for new messages when tab is inactive
- **Visual Status Indicators**: Live/Paused status shown in chat header
- **Optimized Data Transfer**: Uses timestamp-based filtering to reduce bandwidth

### Chat Features
- Global chat room for all users
- Real-time message synchronization
- User profile pictures and usernames
- Message timestamps and date grouping
- Auto-scroll to latest messages
- Responsive design for all devices

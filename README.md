# Portal - Solana-Powered Community Platform

This is the main repository for the Portal project, a modern community platform built with Remix and Solana blockchain integration.

## Project Structure

- `xPortal/` - Main application directory containing the Remix app
- `programs/` - Solana programs (if any)

## Quick Start

1. Navigate to the xPortal directory:
   ```bash
   cd xPortal
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start development:
   ```bash
   npm run dev
   ```

4. Deploy to Cloudflare Workers:
   ```bash
   npm run deploy
   ```
